# 数据结构与算法 PPT

这是一个关于数据结构与算法的PPT集合，包含以下主题：

## 目录

0. **数据结构与算法第一课** (`数据结构与算法第一课.html`)
   - 数据结构与算法的基本概念
   - 数据结构与算法的重要性
   - 数据结构与算法的应用场景

1. **数组 - 内存世界的原住民** (`01_数组_内存世界的原住民.html`)
   - 数组的基本概念
   - 数组的内存布局
   - 数组的常见操作

2. **链表 - 内存的散居游民** (`02_链表_内存的散居游民.html`)
   - 链表的基本概念
   - 链表的内存布局
   - 链表的常见操作

3. **栈 - 只在尾部操作的list** (`03_栈_只在尾部操作的list.html`)
   - 栈的基本概念
   - 栈的内存布局
   - 栈的常见操作

## 使用方法

直接在浏览器中打开对应的HTML文件即可查看PPT内容。
